私たちの想い

まだ、自宅葬が主流だった時代にセレモニー吉岡ではいち早く、葬祭会館を建設しました。
社葬のような大規模な葬儀から家族葬のような小規模葬儀まで幅広く対応しております。
地域の皆様の心を支え続けて三十五年。想いをカタチに…大切な人との最後の時間「心を込めたお見送り」をお手伝いさせて頂きます。
